# Calculator-Python

Simple console-based calculator written in Python.

## Features
- Supports basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Allows multiple consecutive calculations
- Handles division by zero
- Finishes calculation using "=" operator

## Technologies
- Python 3

## How to run
Make sure you have Python 3 installed.

```bash
python calculator.py
